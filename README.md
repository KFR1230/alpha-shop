## Features 功能
1.模擬結帳流程顯示
2.模擬結帳資訊填寫
3.模擬結帳流程控制按鈕
4.目前僅供畫面（無功能）

## Prerequisites 
Node.js 14.16.0
React 18.2.0
Sass 1.62.1

## Installation and Execution
1. 請先確認有無下載node.js 和 npm
2. 將專案clone到本地 
3. 透過終端機開啟資料夾，並輸入
 npm install
4. 安裝完後，輸入
 npm start
5. 打開瀏覽器輸入http://localhost:3000，便可看到畫面
6. 若想停止到終端機輸入
  ctrl + c
